Live2D Widget API 集成测试

说明：此页面用于测试 Live2D API 是否正确适配了 live2d-widget 的格式要求。 live2d-widget 期望以下端点格式：
  • GET /model_list.json - 返回 {messages: string[], models: string[]}
  • GET /model/{modelName}/index.json - 返回模型配置
  • GET /model/{modelName}/textures.cache - 返回纹理缓存（Cubism 2）

1. 模型列表端点测试

GET /model_list.json

2. 模型配置端点测试

GET /model/{modelName}/index.json

3. 纹理缓存端点测试

GET /model/{modelName}/textures.cache

4. Live2D Widget 兼容性检查
